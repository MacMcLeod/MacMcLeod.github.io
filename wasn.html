Network Simulator 3

Network Simulator 3 is a discrete event simulator aimed at educational and research use. NS-3 simulations are written with a C++ library or in Python. Simulated events are simply function calls; callbacks reduce compile-time dependencies between objects. The project has helpful Doxygen documentation but does not provide an IDE.

Introductions to Network Simulator 3: the flow monitor, the hidden terminal problem, and two propagation loss models.

Wireless Networks

Networking is essential for efficiently moving information between nodes. Three common topologies are cellular networks, mobile ad hoc networks, and wireless sensor networks. Metrics for determining the efficiency of transmission are called quality of service (QoS) and include loss, jitter, throughput, etc. These can be controlled by implementing techniques such as admission control, traffic access control, packet scheduling, buffer management, congestion control, QoS-aware routing, and more.
