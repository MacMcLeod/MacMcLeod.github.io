Network Simulator 3

Network Simulator 3 is a discrete event simulator aimed at educational and research use. NS-3 simulations are written with a C++ library or in Python. Simulated events are simply function calls; callbacks reduce compile-time dependencies between objects. The project has helpful Doxygen documentation but does not provide an IDE.

Introductions to Network Simulator 3: the flow monitor, the hidden terminal problem, and two propagation loss models.
